# Coverage Gap Analysis: Prosthetic & Orthotic Care in Underserved US Regions

## Methodology

Geospatial analysis using OpenStreetMap data to identify healthcare provider density in three underserved regions. Search categories included: healthcare, medical, pharmacy, clinic, hospital, rehabilitation. Specific prosthetist/orthotist searches returned zero results across all regions.
